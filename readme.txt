This is a skeleton project that contains the basic elements to start a new web page using perl and sqlite with Templates [Template::Toolkit]
The following directory structure is used:

webpage/
	cgi-bin/
		lib/
		batch/
		web/
	config/
		db.cfg
		tmpl.cfg
		web.cfg
	db/
		data/
	html/
		css/
		img/
	tmpl/
		includes/
